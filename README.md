3. speed-detector.js
a) Description 
The `speedDetector` function prompts the user to enter the speed of a car in km/h, validates the input, and determines whether the speed is within the limit or if demerit points should be given. The following rules are applied: - If the speed is less than 70 km/h, it prints "Ok". - For every 5 km/h above the speed limit (70 km/h), one demerit point is given. - If the total demerit points exceed 12, it prints "License suspended".
b) Usage
1. Install Node.js and the `prompt-sync` package if you haven't already. 
2. Create a JavaScript file and include the code with `prompt-sync`. 
3. Run the script in your terminal with `node <filename>.js`. 
4. Input the speed of the car when prompted.
